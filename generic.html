Escape The Professor

Type

2D Video Game

Language

Python

Framework

Pygame

Genre

Arcade / Pac-Man Style

Escape The Professor is a 2D Pac-Man style game developed in Python using Pygame. The player controls a Student who must navigate university halls to collect all Assignments (Books) while avoiding Professors.

Built entirely in Python, the game features classic arcade mechanics — maze navigation, enemy AI, score tracking — wrapped in a university-themed world that makes it a fun take on the classic formula.

In addition to writing the code, all characters were designed specially for the project using Adobe Illustrator, and the theme soundtrack and sound effects were also custom-made using Ableton.
